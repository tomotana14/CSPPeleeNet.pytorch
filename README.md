| ---- | ---- | ---- | ---- | ---- |
| PeleeNet |  2.802  | 513.876 |-|-|
| CSPPeleeNet (gamma=0.5) | 2.856 |569.063 |73.07|91.11|

Usage
```
import torch

net = torch.hub.load('tomotana14/CSPPeleeNet.pytorch', 'csppeleenet', pretrained=True)
```
